# Campus Bites – COEP Canteen Ordering System 🍔

A sleek and interactive web application that helps students of COEP Tech browse canteen menus, select dishes, and schedule pickup times — all from one clean interface.

## 🌟 Features

- ✨ Stylish, modern UI with hover animations and shadows
- 🖼 Visual display of canteen options (Meta Canteen & Boat Club Canteen)
- 🍽 Dish previews with names, images, and prices
- ⏰ Flexible pickup time slot selection (morning & afternoon)
- ✅ Confirmation system for completed orders

## 🖥 Tech Stack

- *HTML5*
- *CSS3* (no external frameworks, pure CSS)
- *JavaScript* (assumed functionality like showMenu() and selectTime())

## 🚀 How to Run

1. Clone or Download this zip file.
2. Extract it to your PC
3. Open index.html in your browser.
4. Click on any canteen card to view the menu.
5. Select a dish and proceed to choose a pickup time.
6. Confirm your order and you're good to go!
